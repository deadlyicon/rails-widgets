# Rails::Widgets

A view component system for rails

## Installation

Add this line to your application's Gemfile:

    gem 'rails-widgets'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rails-widgets

## Usage

    rails generate widget foobar
